# Customer Segmentation Using Machine Learning

## Overview
This project applies machine learning techniques to segment customers based on purchasing behavior.
Customer segmentation helps businesses better understand their customers and design targeted strategies.

## Problem Statement
Given customer data, the goal is to group customers into meaningful segments using data-driven methods.

## Approach
- Loaded and explored customer data
- Selected relevant numerical features
- Applied K-Means clustering for segmentation
- Visualized customer clusters

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Key Learnings
- Data preprocessing and feature selection
- Unsupervised machine learning using K-Means
- Translating business problems into data science solutions
- Interpreting and visualizing clustering results
